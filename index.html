Chuleta de HTML

Etiqueta Nombre Uso
<header> Header Se usa para diferenciar zonas dentro de la etiqueta body,esta define la cabecera,el menú superior por así decirlo
<main> Main Se usa para diferenciar zonas dentro de la etiqueta body,esta define el cuerpo,hace referencia a todo el contenido de la pagina de interés
<footer> Footer Se usa para diferenciar zonas dentro de la etiqueta body,esta define el pie,el menu inferior por así decirlo
<article> Article Los artículos se usan para diferenciar los contenidos dentro del main, es mejor opción al div
