Estructura

Etiqueta Nombre Uso
<header> Header Se usa para diferenciar zonas dentro de la etiqueta body,esta define la cabecera,el menú superior por así decirlo
<main> Main Se usa para diferenciar zonas dentro de la etiqueta body,esta define el cuerpo,hace referencia a todo el contenido de la pagina de interés
<footer> Footer Se usa para diferenciar zonas dentro de la etiqueta body,esta define el pie,el menu inferior por así decirlo
<article> Article Los artículos se usan para diferenciar los contenidos dentro del main, es mejor opción al div

Margen

Etiqueta Nombre Uso
<width> width/ancho Define el ancho de un elemento dentro de la pagina,puede usarse las medidas como pixel (px) o porcentaje (%)
<height> height/altura Define el alto de un elemento dentro de la pagina,puede usarse las medidas como pixel (px) o porcentaje (%)
<border> border/borde Se usa para modificar la visualizacion del borde de los elementos,se puede especificar el tamaño, el estilo y el color asi como se puede configurar su radio de curvatura (50% hace un circulo completo)
<padding> padding/Sangria Es la distancia de el contenido de un elemento a su borde,puede diferenciarse en 4 posiciones (top,right,bottom,left), se puede definir en una sola linea
<margin> margin/margen Es la distancia de el contenido de un elemento al borde de la pagina,puede diferenciarse en 4 posiciones (top,right,bottom,left), se puede definir en una sola linea
<margin/padding auto> Margen/Sangria auto Si un elemento cuenta con una anchura determinada al aplicarle el margen/sangria auto se centrara en su mitad
<margin/padding Overflow Overflow Determina la visibilidad y estructura de un elemento en caso de que este sea superpuesto en la pagina,se puede configurar para mostrarse con un scroll lateral,mostrarse o ocultarse
<visibility> visibility/Visibilidad Afecta a la visibilidad de un elemento en la pagina, al hacerlo oculto su espacio queda reservado
<Box-sizing> Modelo Box Determina el modelo a usar por la pagina para dimensionar los elemntos,por defecto usa <Box-sizing: content-box> el cual modifica la altura y anchura aplicada en funcion del magen y padding,la solucion es usar <Box-sizing: border-box>
